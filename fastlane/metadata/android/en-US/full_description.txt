<p><i>Mousedroid</i> is a cross platform application that transforms your android phone in an input peripheral. Use it as a mouse (touchpad), keyboard or numpad. Wired or wirelessly, control your pc with your phone.</p><p><br><b>Supported platforms:</b></p><ul><li>Windows</li><li>Linux</li><li>Android 8 (Oreo) and newer</li></ul><p><br><b>Notes:</b></p><ul><li>For Android versions greater than 31 (Android 12L and newer) the app requires bluetooth permission to display the bluetooth name set by the user.</li><li>If you start the server after connecting your phone via USB you need to restart ADB.</li><li>On Wayland some features, like the taskbar icon may not be visible, switching to Xorg might solve the issue.</li><li>For linux the app, development and tests were done on a Ubuntu 22.04 virtual machine. There might be some differences when run on a non-virtual machine intallation</li></ul>
